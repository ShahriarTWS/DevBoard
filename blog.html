Question-1: What are the different ways to select an element in the DOM?

In JavaScript, there are five methods for selecting an element from the DOM. There are:

  • getElementsByTagName()
  • getElementById()
  • getElementsByClassName()
  • querySelector()
  • querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent?

The difference between innerHTML, innerText, and textContent are:

  • innerHTML: Retrieves or modifies HTML elements, such as tags.
  • innerText: Ignores hidden material and only retrieves or modifies the text that is visible.
  • textContent: Retrieves or modifies entire text without formatting, including hidden content.

Question-3: What is event delegation in the DOM?

Event delegation is a JavaScript approach in which a single event listener is attached to a parent element rather than to each child element.

Question-4: What is event bubbling in the DOM?

Event Bubbling is a DOM concept in which an event begins with the target element and goes upwards to its parent elements until it reaches the root element.

Question-5: How do you create, add, and remove elements using JavaScript?

To create, add and remove elements in JavaScript we use:

  • To create an element, use the createElement() method.
  • To add an element, use the appendChild() method
  • To remove elements, use the remove() method or removeChild() on the parent element.
